Cancion

Clase que representa a una canción que será utilizada para ambientar la partida.

Nota: Algunos atributos se pueden dejar "en blanco" (cadenas/vectores vacíos), pero no se pueden guardar de esa forma en el archivo JSON.

new Cancion(nombrePersonalizado: string, ubicacion: string, esLocal: boolean, nombreArchivo: string, nombreOriginal: string, origen: string, tags: Array<string>, descripcion: string, usos: string, prioridad: number)

Extends Persistencia

Parameters
nombrePersonalizado (string) El nombre establecido por el usuario para la canción.
ubicacion (string) Directorio o URL donde se encuentra el audio a reproducir.
esLocal (boolean) Indica si el audio a reproducir se encuentra almacenado de forma local, o en otro servidor.
nombreArchivo (string = "") El nombre con el que está guardada la canción. Si no es local, queda en blanco.
nombreOriginal (string = "") El nombre original de la canción.
origen (string = "") La OST a la que pertenece la canción, o en caso de no tener, el autor.
tags (Array<string> = []) Tags para obtener y clasificar mejor las canciones.
descripcion (string = "") Una breve descripción que indica para qué puede ser usada la canción.
usos (string = "") Breve ayuda visual para ver en qué momento usar la canción. Similar a la descripción pero más breve, y orientado a un uso más específico. Por ejemplo, música del villano.
prioridad (number = "") Ayuda visual. Dependiendo el número se asigna un color a la canción.
