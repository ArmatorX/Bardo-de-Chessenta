Class: Cancion

Cancion(nombrePersonalizado, ubicacion, esLocal, nombreArchivoopt, nombreOriginalopt, origenopt, tagsopt, descripcionopt, usosopt, prioridadopt)

Clase que representa a una canción que será utilizada para ambientar la partida.

Nota: Algunos atributos se pueden dejar "en blanco" (cadenas/vectores vacíos), pero no se pueden guardar de esa forma en el archivo JSON.

nombreArchivo string <optional>
 "" El nombre con el que está guardada la canción. Si no es local, queda en blanco.
nombreOriginal string <optional>
 "" El nombre original de la canción.
origen string <optional>
 "" La OST a la que pertenece la canción, o en caso de no tener, el autor.
tags Array.<string> <optional>
 [] Tags para obtener y clasificar mejor las canciones.
descripcion string <optional>
 "" Una breve descripción que indica para qué puede ser usada la canción.
usos string <optional>
 "" Breve ayuda visual para ver en qué momento usar la canción. Similar a la descripción pero más breve, y orientado a un uso más específico. Por ejemplo, música del villano.
prioridad number <optional>
 "" Ayuda visual. Dependiendo el número se asigna un color a la canción.
